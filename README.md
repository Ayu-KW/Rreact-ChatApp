## 見る時に以下のコードを打ってね！

1.ローカル環境に落とす

```言語:terminal
gitclone
```

2.npm(yarn)で環境を整える

```言語:terminal
npm install
yarn install
```

3.npm(yarn)を立ち上げる

```言語:terminal
npm start
yarn start
```
